# Stockyard Cache

> Response caching for your LLM calls. Same question = instant cached answer, saves 30-50%.

OpenClaw skill that wraps Stockyard's llmcache product.

## Install

Paste this URL in your OpenClaw chat:
```
https://github.com/stockyard-dev/stockyard-cache-skill
```

## Usage

- **cache_stats**: Show cache hit rate and savings
- **cache_clear**: Clear the cache

- **Paid**: unlimited cache, semantic matching, TTL config
