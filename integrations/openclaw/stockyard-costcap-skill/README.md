# Stockyard Cost Caps

> Real-time spend tracking and budget caps. Never get a surprise LLM bill again.

OpenClaw skill that wraps Stockyard's costcap product.

## Install

Paste this URL in your OpenClaw chat:
```
https://github.com/stockyard-dev/stockyard-costcap-skill
```

## Usage

- **spend_query**: Check current spending
- **budget_set**: Set daily/monthly budget
