# Stockyard Router

> Smart model routing. Simple questions → cheap models. Complex reasoning → powerful models. Save 40-70%.

OpenClaw skill that wraps Stockyard's modelswitch product.

## Install

Paste this URL in your OpenClaw chat:
```
https://github.com/stockyard-dev/stockyard-router-skill
```

## Usage

- **routing_stats**: Show routing savings
- **routing_test**: Test which model a prompt would use
