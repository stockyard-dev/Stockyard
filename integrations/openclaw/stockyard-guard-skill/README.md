# Stockyard Guard

> PII redaction and prompt injection detection. Your personal data never reaches the LLM provider.

OpenClaw skill that wraps Stockyard's promptguard product.

## Install

Paste this URL in your OpenClaw chat:
```
https://github.com/stockyard-dev/stockyard-guard-skill
```

## Usage

- **guard_stats**: Show redaction statistics
- **guard_test**: Test text for PII
