# Stockyard Analytics

> LLM usage analytics and reporting. Ask 'show my usage' for cost breakdowns and trends.

OpenClaw skill that wraps Stockyard's llmtap product.

## Install

Paste this URL in your OpenClaw chat:
```
https://github.com/stockyard-dev/stockyard-analytics-skill
```

## Usage

- **analytics_query**: Get usage overview
- **usage_report**: Detailed usage by model
