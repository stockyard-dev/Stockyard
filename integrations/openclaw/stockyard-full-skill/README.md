# Stockyard Suite

> Complete LLM infrastructure in one install. Cost caps, caching, PII redaction, smart routing, analytics, and more.

OpenClaw skill that wraps Stockyard's stockyard product.

## Install

Paste this URL in your OpenClaw chat:
```
https://github.com/stockyard-dev/stockyard-full-skill
```

## Usage

- **stockyard_status**: Show suite status
- **stockyard_spend**: Check spending
- **stockyard_cache**: Cache stats
