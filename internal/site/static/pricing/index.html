Frequently asked
Is self-hosted really free?
Yes. Stockyard is a single Go binary. Download it, run it, use it forever. All six apps, all 50+ modules, no feature gating, no usage limits, no phone-home.
What's the difference between Cloud and self-hosted?
Same software. Cloud means we handle the server, scaling, backups, and uptime. You get a subdomain (your-org.stockyard.dev) and point your app at it. No Docker, no deploys, no ops.
Can I switch from self-hosted to Cloud later?
Yes. Export your config from self-hosted using the Exchange environment sync, then import it into your Cloud instance. Your module toggles, routes, policies, and workflows carry over.
Do you mark up LLM API costs?
No. Stockyard never touches your API keys or charges per token. You pay your LLM providers directly at their prices. Stockyard pricing is only for the platform itself.
What if I need more than 100k requests on Cloud?
The Team tier ($99/mo) covers 1M requests. For higher volumes, Enterprise includes unlimited requests with a dedicated instance.
What counts as a request?
One LLM API call through the proxy = one request. Dashboard views, API management calls, and Exchange operations don't count.
