How it works
1

Download the binary

One command: brew install stockyard-dev/tap/stockyard or npx, docker, or curl.

2

Run it

Set your LLM API key, run the binary. Dashboard appears at localhost/ui.

3

Upgrade when you're ready

Community tier works forever. When you need more requests, modules, or cloud hosting — upgrade and get a license key. Set STOCKYARD_LICENSE_KEY=SY-... and restart.

Common questions
What do I get for free?
The Community tier gives you 1,000 requests/month, 3 providers, 20 middleware modules, and 7-day audit retention. No credit card, no time limit.
What's the difference between Individual and Pro?
Individual is self-hosted — you run the binary on your own server. Pro is cloud-managed — we run it for you with zero ops, auto backups, and 90-day audit retention. Both get all 70 modules and all 16 providers.
Can I save with annual billing?
Yes. Annual plans give you 2 months free: Individual $99/yr (save $21), Pro $490/yr (save $98), Team $1,490/yr (save $298), Enterprise $4,990/yr (save $998).
How does Team pricing work for more than 5 people?
Team includes 5 seats. Additional seats are $25/seat/month. So a 10-person team is $149 + (5 × $25) = $274/month.
How does the license key work?
After purchase, you receive a license key via email. Set STOCKYARD_LICENSE_KEY=SY-... in your environment. The proxy validates it offline using cryptographic signatures — no phone-home, no network dependency.
What happens when my subscription expires?
Your proxy reverts to the Community tier (1,000 requests/month, 3 providers, 20 modules). No data loss, no config changes. Re-subscribe anytime.
What providers are supported?
OpenAI, Anthropic, Gemini, Groq, Ollama, DeepSeek, xAI, Mistral, Cohere, Together, Fireworks, Perplexity, Replicate, Amazon Bedrock, Azure OpenAI, and Google Vertex AI. Any OpenAI-compatible endpoint works out of the box.
Do I need to change my application code?
No. Point your OPENAI_BASE_URL at the Stockyard proxy. Your existing OpenAI SDK calls work unchanged.
What does Enterprise include beyond Team?
Unlimited seats, unlimited audit retention, custom module development, SSO/SAML, SLA with uptime guarantees, compliance export, and a dedicated support channel. Email enterprise@stockyard.dev to discuss.
