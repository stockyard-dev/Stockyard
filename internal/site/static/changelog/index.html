v1.2.0 February 27, 2026 Platform
New Features
Webhooks — register HTTP/Slack endpoints for alerts, cost thresholds, trust violations. HMAC-SHA256 signed. Auto-disable after 10 failures.
API key rotation — POST /api/auth/me/keys/{id}/rotate. Atomic revoke + regenerate, zero downtime.
Status page — /status/ with live auto-refresh. GET /api/status returns uptime, request count, error rate, latency, memory, component health.
Config export/import — snapshot modules, webhooks, and policies. Diff before applying. GET /api/config/export, POST /api/config/import, POST /api/config/diff.
VS Code extension — status bar indicator, module toggle quick-pick, trace viewer, sidebar tree views for modules/traces/providers.
Terraform provider — manage modules, webhooks, and trust policies as infrastructure-as-code.
Docs & Site
Operations docs — doctor, webhooks, OTEL, status monitoring, config export, key rotation.
API docs expanded with Playground, Webhooks, and System endpoint tables.
Integration examples — Python, Node.js, curl, webhook setup.
Sitemap expanded to 38 URLs. Endpoint count updated to 80+.
Infrastructure
Dockerfile (multi-stage, non-root) + docker-compose.yml with healthcheck.
GoReleaser config: unified build, Homebrew tap, Docker image, release notes.
Integration smoke test — builds binary, boots server, hits 40+ endpoints. Runs in CI.
SECURITY.md, .env.example, GitHub issue templates, CONTRIBUTING.md refresh.
Fixes
Fixed playground share button — now calls backend API, loads shared sessions via ?share={id}.
Fixed docs sidebar structure (malformed HTML from nav insertion).
Fixed 404 page favicon paths. Standardized endpoint count to 80+ across all pages.
v1.1.0 February 27, 2026 Platform
New Features
stockyard doctor — Pre-flight environment check (config, database, ports, API keys, Ollama, license, disk). JSON output with --json flag.
OpenTelemetry export — Set OTEL_EXPORTER_OTLP_ENDPOINT to send traces to any OTLP backend (Jaeger, Grafana, Datadog, Honeycomb). Batched export, graceful flush.
Playground share — POST /api/playground/share creates shareable sessions. 30-day TTL, auto-cleanup.
GitHub Action — .github/actions/setup-stockyard for CI/CD pipelines. Install, doctor, start as background service.
Go benchmark suite — 12 benchmarks covering middleware chain, toggle overhead, registry lookup, and parallel throughput.
Docs & Site
Docs split into 10 pages with sidebar navigation, prev/next links, and active state highlighting.
3 comparison pages: vs LiteLLM, vs Helicone, vs Portkey.
Blog with 3 launch posts. RSS feed at /blog/feed.xml.
Interactive architecture diagram at /architecture/ — animated request flow through all 58 modules.
Install script: curl -sSL stockyard.dev/install | sh
Sitemap expanded to 35 URLs. Blog nav link added site-wide.

CI workflow: added benchmark job with artifact upload. Expanded test/vet coverage to engine/ and proxy/.
