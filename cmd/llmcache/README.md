# CacheLayer

**Same prompt? Instant response. 90% cost savings.**

CacheLayer caches identical LLM requests and returns them instantly. Your users get sub-50ms responses, you save 90%+ on repeated prompts, and your app feels magical.

## Quickstart

```bash
export OPENAI_API_KEY=sk-...
npx @stockyard/llmcache

# Dashboard → http://localhost:4101/ui
```

## What You Get

- **Exact-match caching** with configurable TTL
- **Cost savings tracking** — see how much you've saved
- **Latency comparison** — cached (48ms) vs uncached (1,240ms)
- **Cache browser** — inspect and clear entries
- **Zero code changes** — just change your base URL

Part of [Stockyard](https://github.com/stockyard-dev/stockyard).
